# Speckit Workflow

## Core Commands (in order)
1. `/speckit.constitution` - Establish project principles
2. `/speckit.specify` - Create baseline specification
3. `/speckit.plan` - Create implementation plan
4. `/speckit.tasks` - Generate actionable tasks
5. `/speckit.implement` - Execute implementation

## Optional Quality Commands
- `/speckit.clarify` - Structured questions to de-risk ambiguity (before planning)
- `/speckit.analyze` - Cross-artifact consistency check (after tasks, before implementation)
- `/speckit.checklist` - Quality validation checklist (after planning)
